# Logan's Skills

A collection of [Agent Skills](https://agentskills.io/home) for personal productivity and development workflows.

## Skills

| Skill | Description |
|---|---|
| `confluence` | Confluence Cloud REST API — 搜尋、讀取、建立、更新頁面 |

## Installation

```bash
pnpx skills add iml885203/skills --skill='*'
```

Or install a specific skill:

```bash
pnpx skills add iml885203/skills --skill=confluence
```
